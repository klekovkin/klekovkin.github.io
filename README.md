# klekovkin.github.io
Iurii Klekovkin CV

## Repository Structure

- `CV/` - Hugo-based CV source files (git submodule)
- `assets/`, `index.html`, etc. - built site output served by GitHub Pages

## Setup

Clone with submodules:

    git clone --recurse-submodules https://github.com/klekovkin/klekovkin.github.io.git

Or after cloning:

    git submodule update --init --recursive
